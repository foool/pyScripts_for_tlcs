# pyScripts_for_tlcs
python scripts to generate call relationships of TLC files. TLC is an Simulink-defined interpreted Language


## requirements
1. Python3
2. Graphviz & Graphviz for python (pip install graphviz)
3. MATLAB2020 (or other versions)
